🥕 Marco's Store

A fun and interactive web game where users can buy and sell fruits and vegetables using virtual money.



🎮 Features
Buy and sell fruits and vegetables
Real-time balance updates
Quantity tracking for every item
Beautiful product cards with SVG images
Mobile-friendly responsive design
Fast and lightweight
No registration required
Just for fun!
🍎 Available Items

The store includes a variety of produce such as:

Tomatoes
Carrots
Potatoes
Onions
Broccoli
Apples
Bananas
Oranges
Mangoes
Watermelons

...and many more!
